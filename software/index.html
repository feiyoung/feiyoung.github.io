Software

  • ILSE is an R package for Iterative Least Square Estimation or Full Information Maximum Likelihood Estimation for Linear Regression When Data Include Missing Values.
  • nptvcmPD is an R package simplified from NonParametric Time-Varying Coefficient Mmodel for Panel Data (nptvcmPD). Nonparametric time-varying coefficients model based penalty for longitudinal data with pre-specified finite time points, also known as panel data.
